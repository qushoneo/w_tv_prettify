Установка расширения

Проверка на Virustotal

* ВОЗМОЖНО НЕ БУДЕТ РАБОТАТЬ С ЗАПРЕТОМ
  1. Скачать расширение с Github
  2. Установить расширение через настройки расширений
    вот это гамно вставить в адресную строку браузера -> chrome://extensions/
  3. Включить режим разработчика
  4. Загрузить распакованное расширение
  5. Перезагрузить страницу
  6. Теперь смайлики должны работать на w.tv
